MuteTab Options

For more information about each setting, see the MuteTab Documentation.



Autostop exclusions:

Enter one URL per line. If a website's URL matches the beginning of what is entered, it will be considered background music/video and not automatically muted.




Undetected sound sources:

Enter one URL per line. If a website's URL matches the beginning of what is entered, it will always be considered to be a possible sound source. Useful for pages that make use of web audio or popup windows that MuteTab doesn't properly detect.




Sound source preferences

Modify webpages to allow scripting for YouTube, QuickTime, etc. videos where it is disabled (this allows pausing on some sites; only turn off if problems)
